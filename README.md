**FileGuard** жҳҜдёҖдёӘйқўеҗ‘дјҒдёҡзҡ„ AI еўһејәж–Үд»¶и®ҝй—®жҺ§еҲ¶дёҺйҳІжі„йңІзі»з»ҹпјҢеҹәдәҺйӣ¶дҝЎд»»жҖқжғіпјҢжҸҗдҫӣз»ҶзІ’еәҰзҡ„жқғйҷҗз®ЎзҗҶгҖҒеҠЁжҖҒеҠ еҜҶгҖҒAI йЈҺйҷ©иҜ„еҲҶгҖҒиЎҢдёәе®Ўи®Ўе’Ңз»Ҳз«ҜйҳІжҠӨиғҪеҠӣгҖӮ

- **йҖҡз”ЁжҖ§**пјҡйҖӮз”ЁдәҺеҲ¶йҖ гҖҒйҮ‘иһҚгҖҒз ”еҸ‘зӯүд»»дҪ•еҜ№ж•Ҹж„ҹж–Үд»¶жңүдҝқжҠӨйңҖжұӮзҡ„иЎҢдёҡ
- **зӨәдҫӢеңәжҷҜ**пјҡд»Ҙж–°иғҪжәҗжұҪиҪҰдјҒдёҡдёәж ёеҝғжј”зӨәиғҢжҷҜпјҢеұ•зӨәеҰӮдҪ•йҳІжӯўи®ҫи®ЎеӣҫзәёгҖҒз”өжұ й…Қж–№гҖҒдҫӣеә”й“ҫж•°жҚ®зӯүиў«зӣ—жҲ–еӨ–жі„

## иғҢжҷҜдёҺз—ӣзӮ№

дј з»ҹж–Үд»¶и®ҝй—®жҺ§еҲ¶пјҲеҰӮ ACLгҖҒNTFS жқғйҷҗпјүеӯҳеңЁд»ҘдёӢй—®йўҳпјҡ
- **йқҷжҖҒжҺҲжқғ**пјҡжқғйҷҗдёҖж—ҰжҺҲдәҲпјҢй•ҝжңҹжңүж•ҲпјҢж— жі•еә”еҜ№е®һж—¶йЈҺйҷ©
- **еҶ…йғЁеЁҒиғҒ**пјҡзҰ»иҒҢе‘ҳе·ҘгҖҒжҒ¶ж„Ҹз®ЎзҗҶе‘ҳеҸҜжү№йҮҸзӘғеҸ–ж–Үд»¶
- **иҫ№з•ҢеӨұж•Ҳ**пјҡз§»еҠЁеҠһе…¬гҖҒдә‘еҚҸдҪңдҪҝдј з»ҹеҶ…зҪ‘йҡ”зҰ»еҪўеҗҢиҷҡи®ҫ
- **и§„еҲҷеғөеҢ–**пјҡдәәе·Ҙзј–еҶҷзҡ„ DLP и§„еҲҷж— жі•иҜҶеҲ«иҜӯд№үзә§ж•Ҹж„ҹеҶ…е®№

еңЁж–°иғҪжәҗжұҪиҪҰдјҒдёҡдёӯпјҢе…ёеһӢйЈҺйҷ©еҢ…жӢ¬пјҡ
- з ”еҸ‘дәәе‘ҳз§ҒиҮӘжӢ·иҙқж•ҙиҪҰи®ҫи®ЎеӣҫеҲ°дёӘдәә U зӣҳ
- зҰ»иҒҢе·ҘзЁӢеёҲжү№йҮҸеҜјеҮәиҮӘеҠЁй©ҫй©¶ж ёеҝғд»Јз Ғ
- дҫӣеә”е•ҶйҖҡиҝҮеҚҸдҪңйҖҡйҒ“йқһжі•з•ҷеӯҳз”өжұ й…Қж–№ж–ҮжЎЈ

FileGuard йҖҡиҝҮ **еҠЁжҖҒжҺҲжқғгҖҒAI йЈҺйҷ©иҜ„еҲҶгҖҒе…Ёй“ҫи·ҜеҠ еҜҶгҖҒиЎҢдёәе®Ўи®Ў** еӣӣеӨ§ж”ҜжҹұпјҢзі»з»ҹжҖ§и§ЈеҶідёҠиҝ°й—®йўҳгҖӮ

| **е…ЁиЎҢдёәе®Ўи®Ў** | JSON иЎҢж јејҸе®Ўи®Ўж—Ҙеҝ—пјҢи®°еҪ•е®Ңж•ҙеҶізӯ–й“ҫпјҲеҗ«йЈҺйҷ©иҜ„еҲҶпјүпјҢж”ҜжҢҒж–Үд»¶иҝҪеҠ  |

## дҪҝз”ЁиҜҙжҳҺ

### 1. и®ӨиҜҒдёҺзҷ»еҪ•

```bash
# зҷ»еҪ•иҺ·еҸ– JWT Token
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "alice", "password": "password123"}'

# е“Қеә”
# {"success": true, "data": {"token": "eyJhbGciOi..."}}

# и®ҫзҪ® MFAпјҲеҸҢеӣ зҙ и®ӨиҜҒпјү
TOKEN="<д»Һзҷ»еҪ•е“Қеә”иҺ·еҸ–зҡ„token>"
curl -X POST http://localhost:8080/api/auth/setup-mfa \
  -H "Authorization: Bearer $TOKEN"
# иҝ”еӣһ secret е’Ң qrcode_urlпјҢз”Ё Google Authenticator жү«жҸҸ

# йӘҢиҜҒе№¶еҗҜз”Ё MFA
curl -X POST http://localhost:8080/api/auth/verify-mfa \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"passcode": "123456"}'
```

**жөӢиҜ•иҙҰжҲ·пјҡ**

| з”ЁжҲ·еҗҚ | еҜҶз Ғ | и§’иүІ | йЎ№зӣ® |
|--------|------|------|------|
| alice | password123 | engineer | ev_project |
| admin | admin123 | admin | - |

### 2. ж–Үд»¶дёҠдј дёҺдёӢиҪҪ

```bash
TOKEN="<JWT Token>"

# дёҠдј ж–Үд»¶пјҲиҮӘеҠЁ AES-256-GCM еҠ еҜҶеӯҳеӮЁпјү
curl -X PUT http://localhost:8080/file/projects/battery_spec.xlsx \
  -H "Authorization: Bearer $TOKEN" \
  --data-binary @battery_spec.xlsx

# дёӢиҪҪж–Үд»¶пјҲиҮӘеҠЁи§ЈеҜҶпјү
curl -X GET http://localhost:8080/file/projects/battery_spec.xlsx \
  -H "Authorization: Bearer $TOKEN" \
  -o downloaded.xlsx
```

> **и®ҝй—®жҺ§еҲ¶жөҒзЁӢпјҡ** JWT и®ӨиҜҒ вҶ’ ABAC зӯ–з•ҘиҜ„дј° вҶ’ AI йЈҺйҷ©иҜ„еҲҶ вҶ’ DLP жЈҖжөӢ вҶ’ еҠ еҜҶ/и§ЈеҜҶ вҶ’ ж°ҙеҚ°пјҲеҰӮйңҖпјүвҶ’ е®Ўи®Ўи®°еҪ•

### 3. AI йЈҺйҷ©иҜ„еҲҶпјҲеҸҜйҖүпјү

еҗҜз”Ё AI йЈҺйҷ©иҜ„еҲҶеҗҺпјҢGateway еңЁ ABAC йҖҡиҝҮеҗҺдјҡйўқеӨ–и°ғз”Ё Risk Service иҝӣиЎҢе®һж—¶иҜ„дј°пјҡ

```bash
# 1. и®ҫзҪ® Claude API Key
export FILEGUARD_LLM_API_KEY="sk-ant-api03-..."

# 2. зј–иҫ‘ configs/gateway.yamlпјҢеҗҜз”Ё risk
# risk:
```
